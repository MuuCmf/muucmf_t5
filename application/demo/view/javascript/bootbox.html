  2. 模态对话框

Bootbox用来替代系统内置对话框，并于对话框保持一致的外观。

Bootbox 独立组件，你需要从本地或 CDN 单独引入 lib 目录下的资源：

<link href="lib/bootbox/bootbox.min.css" rel="stylesheet">
<script src="lib/bootbox/bootbox.min.js"></script>

具体用法请参考官方网站。
